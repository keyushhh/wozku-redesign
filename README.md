# Wozku Website Redesign

A modern, interactive redesign of the Wozku marketing website built with React, TypeScript, and Vite.

This project reimagines the Wozku experience with a cleaner visual language, improved storytelling, interactive animations, responsive layouts, and an enhanced user experience across the entire website.

---

## ✨ Features

- Modern landing page redesign
- Interactive hero section
- Responsive design
- Animated UI components
- Global reach visualization
- Interactive India map
- Core team showcase
- ROI calculator
- Case studies
- Customer success stories
- Security & Compliance pages
- Pricing page
- Blog pages
- FAQ section
- Dark mode illustrations
- Optimized WebP assets

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

Clone the repository:

```bash
git clone https://github.com/keyushhh/wozku-redesign.git
```

Navigate into the project:

```bash
cd wozku-redesign
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

## 📦 Production Build

Create a production build:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
